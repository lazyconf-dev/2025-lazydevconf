## 2025 게으른 개발자 컨퍼런스

제 2회 게으른 개발자 컨퍼런스에 참여해주셔서 감사합니다 🙌

컨퍼런스에서 안내드린 바와 같이 해당 레포지토리는 발표자료 그리고 데모코드 등을 공유하기 위해서 생성되었습니다.

---

### 세션 1
+ 발표자 : 황보성우
+ 제목 : 24시간 끊기지 않는 잔고 서비스 개발
+ 내용 : 수많은 트래픽 폭격에도 견뎌낸 가상자산 거래소 잔고 서비스 개발기
